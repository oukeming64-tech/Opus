# 允许失败的合同

4月24日，DeepSeek把V4预览版放了出来。

官方页上没有豪言壮语，只有几行技术参数：V4-Pro，1.6万亿总参数，490亿激活参数；全系列默认百万上下文；API当天可用。页面最后还有一句话，说他们仍然相信 longtermism，继续向AGI走。

同一时间，另一张表也在不断更新。天眼查显示，梁文锋的直接持股从1%升到34%，直接加间接合计持有DeepSeek约84.29%。再往后，融资传闻开始一版一版往上滚：100亿美元、200亿美元、450亿美元、515亿美元、590亿美元。国家大基金、腾讯、京东、网易、宁德时代，名字陆续被媒体写进"知情人士称"里。DeepSeek没有官宣，投资人已经在社交平台上卖"份额"，有人喊"明天锁定资金"。

这就是中国AI最有意思，也最疼的一行背景标注。

它先证明了第一个错误答案是错的：中国AI不行式因为中国人不行，不够聪明。R1已经把这句话打碎过一次。2025年1月20日，DeepSeek-R1开源，MIT License，性能直逼OpenAI o1。硅谷当时的惊讶，不是礼貌性或者象征性的的，惊讶反应在股票上也反应在疼朗普对华强硬的态度上，中国不缺聪明人，也不缺能把工程做到极致的人。

第二个错误答案也站不住：钱不够，市场化不够。2026年一季度，国内AI融资近600起，总额超过1100亿元。外币基金也在回暖，虽然145亿元放在5590亿元全市场募资里，占比不到3%，但至少说明钱不是完全断流。月之暗面、阶跃星辰，一个月能拿走300多亿元。钱已经多到不再能解释"为什么总差一口气"。

问题不是钱的总量。

问题是钱背后的合同。

一笔钱进来，背后跟着返投、对赌、上市时间表、地方招商KPI、税收承诺、就业承诺、下一轮估值锚点。它当然也支持研发，但它更喜欢可汇报的研发：论文数，榜单分，客户logo，融资轮次，产业链协同，发布会上的PPT截图。公司被迫优化可见性，而不是能力。能力要在黑屋子里失败十年，可见性最好下季度就能截图。

国家其实知道这个问题。所以才有二十年存续期的国家创业投资引导基金，才反复说"投早、投小、投长期、投硬科技"。这些话也是真诚的。问题是，耐心写在存续期里，焦虑写在返投条款里。嘴上十年磨一剑，手上保留随时收剑的权力。

这不是某个人坏或者是某个制度烂透了。地方要GDP，要税收，要就业，要上市公司；基金管理人要DPI，要退出，要证明自己没有乱花钱；创业者要下一轮，要留人，要买卡；员工要房子，要孩子上学，要把纸面财富换成不会一夜归零的东西。每个人都是理性的。

于是第二篇里的"股权换水泥"，在这里变成了第三篇的答案。

一个顶级AI研究员拿到期权，为什么最后会去看杭州豪宅？因为他不只是在买房，他是在给不确定性上保险。教培行业曾经就是一条旧伤疤：万亿产业，一纸文件，估值重写。在这种环境里，股权的折现率会被打到天上。顶级地段的房子，反而像全国政策风险最低的资产。所谓没格局，很多时候只是对系统不确定性的诚实定价。

DeepSeek原本是这套系统的漏网之鱼。

它不融资，不路演，不急着商业化。幻方的钱在后面供着，梁文锋用量化利润养一群人做基础模型。它的成功，恰恰来自它暂时不属于那个系统。没有返投，没有季度汇报，没有VC在会议室里问商业闭环，没有地方项目表要填。它可以慢，可以怪，可以把钱烧在别人看不懂的地方。

所以R1不是中国AI常规路径的胜利。它是常规路径之外的幸存者。

然后海平面涨了。

V4不是一个"更聪明的聊天机器人"这么简单。1.6万亿参数、百万上下文、Agent能力、企业服务，这些东西都要算力。训练是一笔大开销，推理是一笔长期账单。模型一旦从实验室走向企业服务，钱就不再是"烧一次"，而是每一次调用都在烧钱。DeepSeek可以把V4-Pro API降到原价四分之一，把每百万token输出价压到6元，但价格越低，越说明它必须用规模、数据和市场份额去换未来。

鱼开始学游泳池里的泳姿。

人才也在流动。V4技术报告致谢近300名研发者，其中10人标注已离开团队。媒体梳理里，王炳宣去了腾讯，罗福莉去了小米，郭达雅去了字节Seed，阮翀去了元戎启行，魏浩然也已离职。原因不神秘：大厂能给更高现金，更清楚的期权，更成熟的退出。一个拒绝资本干预的研究体，也会一起拒绝资本最强的留人工具。

这不是梁文锋变坏了。

恰恰相反，4月27日那次股权调整，像一次抵抗。他把直接持股拉高，合计控制约84.29%；融资传闻里，又说他拟个人出资200亿元人民币，占本轮约四成。最诚实的写法不是"DeepSeek变质了"，而是：即便是最成功的漏网之鱼，即便创始人用真金白银死守控制权，也无法再独自负担前沿探索的算力账单。

靠一个怪人、一家量化基金、一套自有资金，去对抗整个AI军备竞赛，本来就是奢侈品。它不能规模化，而且正在到期。

这里也必须承认两个事实。

第一，芯片禁运是真约束了我国ai行业的快速增长。算力差距不能全怪合同结构。美国的出口管制、GPU短缺、国产算力适配成本，都是真实的物理墙。DRAM一季度合约价被集邦上修到季增90%到95%，长鑫科技一季度净利润330亿元，日赚近3.6亿元。模型公司在烧钱，卖铲子的在印钞。这是算力通胀带来的短期红利。

第二，美国也不是田园牧歌。OpenAI一样被算力推向巨型资本。它2019年改成 capped-profit 时，自己就说未来需要几十亿美元投入云计算、人才和AI超算。后来又改成PBC，非营利基金会继续控制营利实体。美国也焦虑，也商业化，也被巨额资金裹挟。

但差别在工具成熟度。

同样是"保护创始人继续做研究"的权利，美国可以写进格式条款：无投票权股、双层股权、PBC、capped-profit、不带对赌的巨额轮次。中国的梁文锋，只能用200亿元现金现场赎买。

中美AI的差距，不是智力差距，也不只是技术差距。

它是资本时间视界的差距。是钱的合同允不允许失败十年的差距。

三篇文章到这里闭上了。第一篇里，考试说要找牛顿，结果把十八年的家庭投入显影出来。第二篇里，国家把钱引向硬科技，钱穿过估值、期权和退出，最后沉进豪宅。第三篇里，真正接近牛顿的人出现了，但他也要面对同一个系统：融资、估值、股权、算力、人才、退出。

我们不是缺牛顿，我们缺一份允许牛顿失败十年的合同。
