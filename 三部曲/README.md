# 三部曲

> 我们不是缺牛顿，我们缺一份允许牛顿失败十年的合同。

三篇文字，一条线：一个社会如何寻找天才，又如何在寻找的路上，把天才需要的土壤，提前明码标价。从六月的高考考场，到杭州验资八千万的售楼处，再到 DeepSeek 的股权表——同一套系统的三张面孔。

1. **[高考的显影液](1-高考的显影液.md)** —— 考试说要找牛顿，显影出来的却是十八年的家庭投入。
2. **[牛顿的学区房](2-牛顿的学区房.md)** —— 国家把钱引向硬科技，钱穿过估值与期权，最后沉进豪宅。
3. **[允许失败的合同](3-允许失败的合同.md)** —— 真正接近牛顿的人出现了，他面对的还是同一套系统。
